# confur-api

### Quickstart

```
cd $GOPATH/src
git clone https://github.com/confur-me/confur-api.git
cd confur-api
make
make install
confur -c=/path/to/config/file.yml start
```

### Development

```
cd $GOPATH/src
git clone https://github.com/confur-me/confur-api.git
cd confur-api
go run confur.go start
```
